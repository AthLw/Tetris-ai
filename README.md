# Tetris Game

## 注意！ 实现ai需要安装node.js

## JavaScript files

> 1. alllisten.js  
    添加监听器
> 2. lib.js  
    执行过程中需要用的库函数
> 3. Model.js  
    方块的定义
> 4. render.js  
    渲染函数
> 5. Tetris.js  
    游戏主要文件，定义用到的各项数据以及函数
> 6. Thread.js
    游戏入口
